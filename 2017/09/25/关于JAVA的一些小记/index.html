1.线程调用 run 和 start 的 区别

调用run方法，其本质就是一个普通方法的调用，不会开启新的线程。

调用start方法，才会开启一个线程来执行run里面的方法

2.使用Thread和Runnable的区别

Thread：先继承Thread，然后通过new来实现类

Runnable：先继承Runnable，然后new Thread(new 实现类的对象,线程名)
