我与Scala的心路历程

2018-01-21

Scala是一个多写一个分号就是多余的艺术语言

所有Scala编译后的文件都是 *.class 后缀，也就是说，它完全运行在Java虚拟机中。

一切的开始—定义变量

1.变量用var定义就是可变的，相当于java中的普通变量。用val定义是不可变的，相当于java中final类型定义的变量

2.变量所有的类型不必声明，由值来决定

3.Scala有7种数值类型，和Java一样，但是没有包装类型

程序的灵魂—分支语句

Java中的if else 和elseif 完全适用于Scala中,if else的判断后可以用一个值来接，一共有两种方式

1.一句撸

2.块表达式

重复的工作—循环

Scala中有for和while、do while循环

在scala中循环变量的迭代用to表示

for循环

1. <- 相当于=，把迭代中的值赋给i

2. 如需定义多个变量，每个变量之间有一个”;” 在最后判断语句中不用

3. yield的作用是。。每迭代一次，就把i的值取出来做yield后面的操作

Scala的烧脑之处—函数与方法

函数在Scala中占着重要的地位，函数与方法类似，都是传值进去然后进行操作。不同的是，函数可以作为一个变量来使用

定义函数

定义函数的关键在于 => 这也是判断一个函数和方法的重要依据。函数的返回值可以省略（递归中不可省略），变量也可以省略（匿名函数）

定义方法

定义方法的关键在于 def和= 。方法中返回值也可以省略，方法不能作为变量传入，只能调用

方法与函数的结合(烧脑部分)

程序首先定义了一个传入一个函数的方法，方法的内容就是往传入的这个函数里面写两个Int值

接着再定义一个函数，内容是将传入得到两个参数相加

最后把定义好的函数传入方法中！

下划线的妙用

符号 _ 可以将方法转化为函数（原方法不变）也可以将系统中println等转化成函数

数的集合—元组、数组、集合、映射

数据和映射以及集合都包括定长和变长，只是两个包的事，就是

1.数组

2.映射

3.元组

映射是特殊的对偶元组，元组的下角标是从1开始的，元组里可以存多种类型

4.List

符号::是右结合的，也就是1 :: 2 :: 3相当于 1 :: (2 :: 3 )

5.Set

Set是一个无重复元素，并且有序的集合
